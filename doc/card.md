# Card 卡片
可承载文字、列表、图片、段落，常用于后台概览页面。

## API

| 参数 | 说明 | 类型 | 默认值 | 版本 |
| :-   | :-   |:- | :- |  :-: |
| title | 标题内容 | Any | - |
| titleStyle | 标题内容 | Any | - |
| extra | 标题栏右侧的内容 | Any | - |
| bodyStyle | 内容的样式 | Object/String | - |
| footer | 底部内容
| footerStyle | 底部的样式 | Object/String | - |
| bordered | 是否显示边框 | Boolean | false |
| width | 卡片的宽度 | Number | - |
| height | 卡片的高度 | Number | - |
| size | 内边距值 | String | - |
