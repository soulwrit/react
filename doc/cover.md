# Cover 图像封面

## API

| 参数 | 说明 | 类型 | 默认值 | 版本 |
| :-   | :-   |:- | :- |  :-: |
| name | 组名称 | String | `Date.now()` |
| model | 数据提取模型 | Model | undefined |
| onChange | 选中状态发生变化时执行的函数 | Function | noop | undefined |
---
**注** 组中的`checkbox`将会集成组名称,单个`checkbox`无需指定名称
